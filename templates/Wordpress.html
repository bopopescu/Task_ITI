Wordpress

WordPress (WordPress.org) is a content management system (CMS) based on PHP and MySQL[4] that is usually used with the MySQL or MariaDB database servers but can also use the SQLite database engine.[5] Features include a plugin architecture and a template system. It is most associated with blogging but supports other types of web content including more traditional mailing lists and forums, media galleries, and online stores. Used by more than 60 million websites,[6] including 33.6% of the top 10 million websites as of April 2019,[7][8] WordPress is the most popular website management system in use.[9] WordPress has also been used for other application domains such as pervasive display systems (PDS)
